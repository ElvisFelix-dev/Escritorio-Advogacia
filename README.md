## Iniciar

1. Clonar o repo
   ```sh
   git clone https://github.com/ElvisFelix-dev/Escritorio-Advogacia
   ```
2. Abra a pasta
   ```sh
   cd Escritorio-Advogacia

3. Instalando NPM packages
   ```sh
   npm install
   ```

4. Iniciar
   ```sh
   npm run dev
   ```
